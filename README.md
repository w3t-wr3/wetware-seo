# Wetware SEO

SEO data collection, scoring, and branded report generation for Wetware Labs.

## Install

```bash
npx skills add w3t-wr3/wetware-seo
```

## Also required

```bash
npx skills add w3t-wr3/wetware-docs
```

The SEO skill uses the Wetware Labs document template from wetware-docs for branded reports.

## Usage

Open Claude Code and say:

- "Run SEO collection on our client sites"
- "Generate an SEO report for Big Thunder Marine"
- "Show me the SEO dashboard data"
- "What's our portfolio WSS score?"

Claude will read the skill and handle everything.
